## Recording Tips

### Before Recording:
- [ ] Clean desktop/browser
- [ ] Close unnecessary applications
- [ ] Prepare sample CSV file
- [ ] Test all features work
- [ ] Have both apps running
- [ ] Clear any previous data if starting fresh

### During Recording:
- Speak clearly and at moderate pace
- Show cursor movements clearly
- Pause briefly between major actions
- Don't rush - quality over speed
- If you make a mistake, pause and restart that section

### Screen Recording Settings:
- **Resolution**: 1920x1080 (Full HD)
- **Frame Rate**: 30 fps minimum
- **Audio**: Clear microphone, reduce background noise
- **Recording Area**: Full screen or focused on applications

### Software Recommendations:
- **Windows**: OBS Studio, Camtasia
- **Mac**: QuickTime, ScreenFlow, Camtasia
- **Linux**: OBS Studio, SimpleScreenRecorder

### Editing:
- Trim dead space at beginning/end
- Add captions if needed
- Include title card with project name
- Add conclusion slide with GitHub link
- Export in MP4 format (H.264)
